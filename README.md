# Basic_HTML_Website
 Projeto do roadmap.sh

 Nesse projeto foi criado um site simples somente com html e css, que contem as seguintes paginas.
<ul>
<li>Página inicial</li>
<li>Projetos</li>
<li>Artigos</li>
<li>Contato</li>
</ul>
 
 Esse projeto cumpre os requisitos que o desafio pede.

     Estrutura HTML semanticamente correta.

    Várias páginas com uma barra de navegação.

    Meta tags de SEO na cabeça de cada página.

    A página de contato deve ter um formulário com campos como nome, e-mail, mensagem etc.
